# PrimeTrade (Full Stack)

This project is a full-stack application built as part of the PrimeTrade Internship Assignment.

It includes:
- A secure and scalable REST API backend with JWT Authentication & Role-Based Access Control.
- A simple React frontend UI to interact with APIs.
- Swagger documentation and clean modular structure.

---

## 🚀 Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt Password Hashing
- Swagger Docs
- Helmet, Rate Limiting, Mongo Sanitize

### Frontend
- React.js (Vite)
- Tailwind CSS (v3)
- Axios
- React Router DOM

---

## ✅ Features Implemented

### Backend Features
✅ User Registration & Login  
✅ Password hashing (bcrypt)  
✅ JWT Authentication  
✅ Role-Based Access Control (user/admin)  
✅ Task CRUD APIs (secondary entity)  
✅ API versioning (`/api/v1`)  
✅ Input validation & sanitization  
✅ Proper error handling and status codes  
✅ Swagger API Documentation  
✅ Secure headers + rate limiting  

### Frontend Features
✅ Register/Login UI  
✅ JWT protected dashboard  
✅ Create / Update / Delete Tasks  
✅ Display tasks list  
✅ Responsive UI (mobile + desktop)  
✅ Shows API success/error messages
