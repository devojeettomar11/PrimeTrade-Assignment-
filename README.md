# PrimeTrade Internship Assignment (Full Stack)

This project is a full-stack application built as part of the PrimeTrade Internship Assignment.

It includes:
- A secure and scalable REST API backend with JWT Authentication & Role-Based Access Control.
- A simple React frontend UI to interact with APIs.
- Swagger documentation and clean modular structure.
