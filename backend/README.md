# PrimeTrade Backend API

## Features
- User Registration & Login
- JWT Authentication
- Password Hashing (bcrypt)
- Role-Based Access (user/admin)
- Task CRUD APIs
- Swagger Documentation
- Validation & Error Handling
- Security Middleware (Helmet, Rate Limiting, Mongo Sanitize)
